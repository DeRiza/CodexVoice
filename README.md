# CodexVoice

CodexVoice 是一个面向 macOS 的本地语音输入工具，目标是让用户可以用快捷键录音、本地转写，并把文字输入到当前正在使用的应用中。

项目目前处于 MVP 迭代阶段，重点是先跑通一条稳定、轻量、可验证的本地语音输入闭环。

## 核心功能

- 本地语音转文字，优先服务中文输入场景
- 全局快捷键启动和停止录音
- macOS 菜单栏常驻，显示基础运行状态
- 使用本地 Whisper 转录，不依赖云端语音服务
- 通过剪贴板粘贴方式把转写结果注入前台应用
- 简洁浮窗反馈录音、处理中、成功和错误状态

## 当前重点

CodexVoice 目前优先追求稳定的 MVP：

1. 热键触发录音
2. 本地完成转写
3. 自动输入到当前应用
4. 清晰处理权限、麦克风、模型不可用等错误

高级功能，例如流式输入、个人词库、语义标点、多页设置面板和打包发布，会在 MVP 稳定后再逐步评估。

## 适合谁

这个项目适合希望在 macOS 上使用本地语音输入、重视隐私、并希望减少云端语音服务依赖的用户和开发者。
